# Neumorphism Login Form

Một giao diện đăng nhập tối giản theo phong cách **Neumorphism**, tập trung vào trải nghiệm người dùng và hiệu ứng CSS thuần.

# Công nghệ sử dụng
* **HTML5** - Cấu trúc ngữ nghĩa (Semantic HTML).
* **CSS3** - Flexbox, Animation, Custom Properties (Variables).
* **Boxicons/FontAwesome** - Hệ thống icon.
* **Google Fonts** - Font "Poppins" & "Sen".

# Tính năng nổi bật
* **Thiết kế Neumorphism:** Hiệu ứng đổ bóng kép (soft UI) tạo độ sâu.
* **Responsive:** Hoạt động tốt trên mọi kích thước màn hình.
* **Interactive:** Hiệu ứng hover, focus và active chân thực cho các nút và ô input.
* **Validation:** Xử lý form cơ bản bằng thuộc tính HTML5 (`required`, `type="email"`).

# Cách chạy dự án
1.  Tải (Clone) project về máy.
2.  Mở file `index.html` bằng trình duyệt bất kỳ.
3.  Hoặc sử dụng **Live Server** trên VS Code để xem thay đổi thời gian thực.
- Để đổi màu chủ đạo, chỉ cần sửa biến trong `:root` ở file `style.css`.
- Các hiệu ứng đổ bóng được tính toán theo tỉ lệ chuẩn để tránh làm mờ giao diện.
